## ๐ About The Project
**Battle of Tanks** is a top-down 2D shooter developed originally in **2021 (1399 SH)** as my university final project. The game features fast-paced tank combat, 2D physics, and stylized visual effects.

Recently (2025), I revisited this project to upgrade the Unity version, fix legacy compilation errors, and add **cross-platform support** (WebGL & Android) with a new input system.

### โจ Features
* **Endless Single-Player Mode:** Fight against waves of enemy tanks. The difficulty increases dynamically (more enemies) as you win each round.
* **Interactive Environment:** Includes environmental hazards like machine guns.
* **Visuals:** 2D stylized art with fantasy shooting effects and explosion animations.
* **Cross-Platform Controls:** Automatically switches between Keyboard/Mouse (PC) and On-Screen Joysticks (Mobile).
